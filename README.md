# dotfiles

Best not to install these blindly.

## Installation

```
wget -qO- https://raw.githubusercontent.com/xyder/dotfiles/master/install_nix.sh | bash
```

or

```
curl -sL https://raw.githubusercontent.com/xyder/dotfiles/master/install_nix.sh | bash
```
